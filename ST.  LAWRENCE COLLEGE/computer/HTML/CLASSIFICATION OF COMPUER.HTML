CLASSIFICATION OF COMPUTERS

  1. Classification of computer on the basis of working mechanism
      a.Analog computer
      Analog computer are the computer which uses continuous data
      b.Digital computer
      Digita computer are the computer which operates on digital data
      c.Hybrid computer
      Hybrid computer are the computer which have both functionality of analog and digital computer
  2. Classification of computer on the baisi of speed/size
    1. Super computer
    2. Mainframe computer
      • Dumb terminal
      • Intelligent terminal
    3. Mini computer
    4. Micro computer
      5. Personnal computer
      Personnal computer is single user computer which uses CPU based on CISC(Complex Instruction Set Computer)
      Workstation computer
      Workstation computer is multiple user computer which uses CPU based on RISC(Reduced Instruction Set Computer)
  3. Classification of computer on the basis of brand
    1. IBM computer
    2. IBM compactible computer
    3. Mac computer(Apple)
  4. Classification of computer on the basis of model
